# 会議読み込み（AIツール → 会議モード）

OKR の②セッション記録とは別に、**普段の会議メモから ToDo・タスクを引き出して登録する**ための機能です。

## いつ使う？

- プロジェクトの定例で出た「やること」をすぐ登録したい
- ブレストの議事メモから次のアクションを抜き出したい
- OKRサイクル外の打ち合わせの結果を反映したい

> OKR の チェックイン／ウィンセッション は **OKRモードの②セッション**を使ってください。役割が違います。

## 使い方

1. サイドバー上部 **「✨ AIツールを開く」** → モードを **「🎙️ 会議」** に切り替え
2. 議事メモを貼り付け／ファイル添付（PDF・Word・VTT・画像 OK）
3. **「タスクを抽出」** をクリック
4. AI が候補（PJ・ToDo・タスク）を提案
5. 候補をレビューして、登録するものにチェック
6. **「登録する」** で plan-app に反映

## 文字数制限

入力は **30,000 文字**まで。長文の場合：
- 重要な部分だけ抜き出して貼り付け
- またはファイル添付（PDF/Word）にして本文は要約だけ
- VTT はそのまま OK（Claude がタイムスタンプを無視）

## よくある質問

### 抽出されたタスクが多すぎる
チェックを外して必要なものだけ残せます。1回で20件超になる場合、議事メモが冗長な可能性。

### 担当者が空欄になる
議事メモに「誰が」が明示されていないことが多い。**手動で担当を割り当ててから登録**で OK。

### PJを跨いでタスクが生まれる
1回の登録で複数PJに振り分けることも可能。会議で議論したPJが複数あれば自動分類されます。
