# ダッシュボード（計画モード）

計画モードのトップ画面。**自分のタスク・PJ全体・期限アラート**を1画面で把握できます。

## 主な構成

| エリア | 内容 |
|---|---|
| **ProjectKarte（PJ選択時）** | サイドバーで PJ を選ぶと、そのPJの要約カードが表示される。AI分析もここから |
| **今週の自分のタスク** | 担当者＝自分・期日が今週のタスクを一覧 |
| **期限アラート** | 期日切れ・期日間近のタスクの抜粋 |
| **PJ進捗カード** | 各PJのタスク達成率・状態の集計 |

## PJ別 AI 分析

ProjectKarte の **「AI分析」** ボタンで、そのPJの直近状況を AI が要約。

- 1 PJ あたり**最新2件まで**を Supabase に保存（共有）
- 「最新の分析」は全員に同期表示
- 旧分析は履歴として確認可

## サイドバー連動

- **「自分のPJのみ」フィルタ**（サイドバー上部）でPJ一覧を絞り込み
- PJ を選ぶとダッシュボードの ProjectKarte ＋ 各セクションがそのPJに連動
- フィルタを外すと全PJを横断表示

## ガント／カンバン／リストとの関係

ダッシュボードは**要約と入口**です。詳細操作は：
- **ガント**：期間と期日のカレンダー表示
- **カンバン**：ステータス列のドラッグ&ドロップ管理
- **リスト**：絞り込み・CSV出力・大量編集
