# ① 会議ノート

KR代表が、KR会議（週次）で記入するノートです。**KR×週で1件**、その中に対象クォーターの全 TF のセクションが並びます。

## 目的

- その週の **仮説・事実・次の一手・進捗％** を TF ごとに記録する
- 翌週の **②セッション** で AI に渡す材料を整える
- ③レポートで確定された前週の学びを **引き継ぎメモ**で受け取り、今週の議論の前提にする

## 基本操作

1. **KR を選択**：上部「Key Result」セレクタから対象 KR を選ぶ。選ぶまで TF は表示されません。
2. **クォーター**：通常は「今」のままで OK。過去 Q の TF も見たい時だけ変更。
3. **対象週**：カレンダーで日付を選ぶと、その週の月曜に揃います。
4. **引き継ぎメモ**（任意）：前週ノートから「↩ 前週から引き継いで作成」、または「↻ 引き継ぎメモを自動生成」で前週の確定レポート＋最新③分析の要点を取り込めます。
5. **TFカード**を上から順に記入。畳んだ TF はクリックで再展開できます。
6. 下部の「保存」または「このKRノートを作成」で保存。

## 各TFカードに何を書くか

| 欄 | 書くこと |
|---|---|
| **TFの説明・テーマ** | そのクォーターの大目的（OneNote の「★1Q＝…」相当） |
| **必達の定義** | この月に到達したい状態 |
| **評価観点** | 何をもって達成と見るか |
| **① 先週動かした前提・仮説** | 仮説ベースで何を動かしたか |
| **② 実際に起きたこと（事実・反応）** | **評価・解釈を入れない**。事実のみ。 |
| **③ 次にやる一手（判断）** | 次の手 |
| **④ 現在のプロセス状態（％＋理由）** | 数値感覚と、その％の根拠 |
| **▶ TODO** | その時期のToDo |

## よくある詰まりどころ

- **TFが表示されない** → クォーター割り当てが未設定の可能性。管理画面でその Q の TF 割り当てを行うか、ガイダンスバナーに従ってください。
- **「保存」が押せない** → KRが未選択、もしくは TF が0件。
- **引き継ぎメモが空** → 前週の③レポートがまだ確定されていない可能性。`↻ 引き継ぎメモを自動生成` を試してください。
