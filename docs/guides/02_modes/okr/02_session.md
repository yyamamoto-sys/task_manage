# ② セッション記録 & 分析

KR1・KR2 合同で行うチェックイン／ウィンセッションの議事メモを取り込み、
AI に **詳細分析 → KRごとに振り分け → 宣言抽出**を一気にやってもらうステップです。

## 目的

- 1時間の会議で出た内容を**取りこぼさず**KRごとに整理する
- 「先に宣言を抽出してから分析」ではなく「**先に詳細分析してその中で宣言も拾う**」順序にすることで、KR1件あたり1宣言しか出ない、という事態を防ぐ
- 結果（分析・宣言）は DB に保存され、③レポートと翌週①の引き継ぎメモに自動連動

## セッションの種類（上部バー）

- **チェックイン／ウィンセッション** — 1回の議事メモを貼って AI が対象KRごとに自動振り分け。
  対象KRは複数選択できるが、1つだけチェックして単一KRの深堀り議論として記録することもできる
- **その他のOKR議論（自由形式）** — 戦略会議・四半期計画などメインKR1つに紐づける freeform 記録

## 基本フロー

1. **対象 KR を選ぶ**（複数選択可）
2. **議事メモを取り込む** — 以下のいずれかで OK：
   - テキストを直接貼り付け
   - **VTT**（Teams 文字起こし）ファイル
   - **Word（.docx）**
   - **PDF**
   - 画像
3. 画面右の **「AI に分析させる」**（スクロール無しで押せる位置に固定）をクリック
4. AI が以下を生成：
   - 会議全体の分析（外部環境・前提変化・論点）
   - **KR ごと**に：シグナル（🟢🟡🔴）／コメント／議論サマリ／学び／リスク／次のアクション／**宣言**（誰が・何を・いつまでに）／**前週宣言の結果**
   - **Objective 全体**の分析（横断的な示唆）
5. 結果をレビューし、必要に応じて手修正
6. **保存** → kr_sessions / kr_declarations / okr_analyses（kr / objective）が作られる

## セッション種別

| 種別 | タイミング | 主な記録 |
|---|---|---|
| **チェックイン** | 週初 | 今週の宣言（誰が・何を・いつまでに） |
| **ウィンセッション** | 週末 | 先週宣言の結果（達成／一部達成／未達）と学び |
| **freeform** | 任意 | 自由形式OKR議論 |

## ファイル取り込みの注意

- **VTT** は Teams のレコーディングから出てくるそのままでOK
- **Word/PDF** は本文テキストが抽出される。表は崩れる可能性あり
- 画像は Claude が直接読む（OCRに近い精度）

## よくある詰まりどころ

- **「AIに分析させる」が出てこない** → 議事メモが空 or 対象KRが未選択
- **KR1件しか宣言が出てこない** → 議事メモが極端に短い場合のみ起こる。1時間分以上であれば複数出るはず
- **VTTの大量タイムスタンプが邪魔そう** → Claudeはタイムスタンプを無視して内容を抽出するので問題なし
- **保存後に直したい** → 履歴オーバーレイ（右上🕐履歴）から該当セッションを開いて編集
