# 初日にやること

plan-app を初めて触る方の **30分セットアップ**ガイドです。順番にやれば、明日からそのまま使い始められます。

## ステップ1：ログインする（2分）

1. ブラウザで plan-app の URL を開く（山本さんから共有されたもの）
2. **メンバー選択画面**で自分の名前を選ぶ
3. 一度選ぶとブラウザに記憶されます（次回から自動）

> 違う名前を選んでしまったら、画面右下の **⏏ ログアウト**ボタンで戻れます。

## ステップ2：全体像を掴む（5分）

2つのモードを行き来して使うアプリです。サイドバー左上の「📋 計画 / 🎯 OKR」で切り替え。

| モード | 主な用途 |
|---|---|
| **📋 計画** | 日々のタスク・ダッシュボード・ガント・カンバン・リスト |
| **🎯 OKR** | 週次サイクル（会議ノート→セッション→レポート）・なぜなぜ・クォーター計画 |

詳しくは [plan-app の全体像（5分）](../00_overview.md) を読んでください。

## ステップ3：自分の関わるプロジェクト／KRを確認する（5分）

### 計画モードで
1. サイドバーの**プロジェクト一覧**で、自分が関わるPJをクリック
2. **ダッシュボード**で進捗を確認
3. 「自分のPJのみ」フィルタはサイドバー上部のトグルで切り替え

### OKRモードで
1. サイドバーの **🎯 OKR** に切り替え
2. 右上の **🎯 OKR** ボタン（履歴の隣）から概要を開く
3. 自分の所属する KR をクリックして選択 → 作業画面に戻ると今週のサイクル進捗が表示される

## ステップ4：自分のタスクを見る（5分）

1. 計画モード → **ダッシュボード**で「今週の自分のタスク」を確認
2. または **カンバン**で未着手／進行中／完了をドラッグ&ドロップ
3. **リスト**で絞り込み・CSV出力もできる

タスクをクリックすると編集モーダルが開き、ステータス・期日・担当者などを変更できます。

## ステップ5：困った時の場所を覚える（3分）

1. **各画面右上の `?` ボタン** — その画面の使い方が即表示されます
2. **サイドバー下部の📖 ガイド** — このガイド全体の目次
3. [FAQ](../06_troubleshooting/faq.md) — よくある詰まり
4. それでも解決しなければ山本さんに連絡

## ステップ6：自分の役割の専用ガイドを読む（10分）

あなたの役割によって、追加で読むページが変わります：

| あなたは… | 次に読むページ |
|---|---|
| KR代表（KR会議でノート記入する） | [KR代表ガイド](../03_roles/kr-rep.md) |
| ファシリ（チェックイン・ウィンを進行する） | [ファシリテーター・ガイド](../03_roles/facilitator.md) |
| 管理者（Objective/KR/TF登録・メンバー管理） | [管理者ガイド](../03_roles/admin.md) |
| TFメンバー（タスクを動かす） | このページで終わり。あとは慣れていけばOK |

## ステップ7：今週の最初の操作を1つだけやってみる

実際に手を動かすのが一番早いです。役割ごとにおすすめの初手：

- **TFメンバー** → 自分のタスクの状態を1つ「進行中」に変えてみる
- **KR代表** → OKRモードの **① 会議ノート** で今週分を1KR入力してみる
- **ファシリ** → OKRモードの **② セッション記録&分析** タブを開いてどんな画面か眺める
- **管理者** → サイドバー下部の **⚙ 設定** で Objective・KR・TF・メンバー登録を確認

これで今日は終わりで OK です。明日以降は実際の業務に合わせて使ってください。

## 1週目の目標

- 自分のタスクを毎日触る（状態更新でOK）
- 自分が KR 代表なら、今週の会議ノートを完了させる
- 詰まったら `?` ボタン

困ったら遠慮なく山本さんに連絡を。
