# FAQ・よくある詰まり

## 全般

### ログインできない / 自分の名前が一覧に無い
管理者（山本さん）にメンバー登録をしてもらってください。
**設定 → メンバー管理**から登録するとすぐ反映されます。

### 画面が真っ白／古いままで動かない
一度 **ブラウザのリロード（Ctrl+R / Cmd+R）** を試してください。
新しいデプロイ直後に、古いチャンクが消えていてエラーになることが稀にあります（自動回復するようには作っていますが、リロードが最速）。

### モバイルでも使える？
基本は PC 向けです。モバイルでも開けますが、OKRモードの分析画面など一部の作業は PC で行ってください。

## 計画モード

### 自分のタスクが見当たらない
- サイドバーの **「自分のPJのみ」フィルタ**がオンになっている可能性があります（既定はオン）
- ダッシュボードでは「自分のタスク」セクションで自分担当分だけ抽出できます
- リストで担当者フィルタを使うのも有効

### タスクの担当者を変えると過去のタスクの動きが変わって見える
タスクの履歴は変更ログとして残ります。ガント・カンバン・リストの3つに表示されますが、すべて同じデータを見ています。

### CSV出力したい
リストビュー → 右上の CSV出力ボタンから。

## OKRモード

### 会議ノートでTFが表示されない
- **KR を選択していない**：上の「Key Result」セレクタから選ぶと、その KR の今クォーターの TF が表示されます
- **クォーター割り当てが未設定**：管理画面でその Q の TF 割り当てが必要です（バナーで案内が出ます）
- 暫定で表示したい場合：「クォーター」セレクタを変えて他のQで存在するTFかを確認

### 会議ノートで「保存」が押せない
- KRが未選択、もしくは TF が0件、もしくは変更なしの状態です
- KR を選び、いずれかのフィールドに記入すると押せるようになります

### セッションで1KRしか宣言が出てこない
- 議事メモが短すぎる可能性（数行レベル）。1時間分の文字起こしであれば複数出るはず
- それでも出ない場合は、「分析結果」を確認し、不足分を手追加で構いません

### VTT / Word / PDF をアップロードしたい
- セッション記録のファイル添付エリアから直接ドロップ
- VTT は Teams のレコーディング書き起こしそのまま OK
- Word は本文テキストが抽出される（表は崩れる可能性）
- PDF は Claude が直接読む

### レポートを確定したのに翌週の引き継ぎメモに出てこない
- **対象週が同じKRの「直前の週」になっているか**確認してください
- 「↻ 引き継ぎメモを自動生成」ボタンで再生成できます
- それでも出なければ、確定が `finalized` ではなく `draft` のままの可能性

### クォーターが変わった時の手順は
1. **クォーター計画**タブで前Qの全データを分析→対話形式で次QのTFを設計
2. 管理画面で次Qの **TF割り当て**を確定
3. 翌週から①会議ノートで通常通り運用

## AI機能

### AIの使用量は確認できる？
設定（⚙）→ AI使用量パネルで、intent ごとの消費トークン・回数を確認できます。

### AIに渡される範囲はどこまで？
OKR運用に必要な範囲（O・KR・TF・ToDo・タスク・会議ノート・分析・宣言・contribution_memo）はすべて渡してOKという方針です（2026-05-13以降の方針改定）。
個人の私的メモなど明示的に「AIに渡さない」とした項目だけ除外されます。

### AIの返答が遅い／止まる
- 大きいファイル（数十ページのPDF・1時間以上のVTT）を投げると 30〜60秒かかります
- 1分待っても応答しない場合、リロードして再実行してください
- それでも続くなら山本さんに連絡

## エラー表示

### `PGRST205` — テーブルが見つからない
DBスキーマがまだ反映されていない状態です。山本さんに伝えてください。

### `42501` — permission denied
DBアクセス権限の不足。エラー本文に修正用 SQL が入っているはずなので、山本さんに転送してください。

### `dynamically imported module` のエラー
新しいデプロイ直後に古いチャンクを読みに行った状態。**リロード**で解消します。

## それでも解決しない時

山本さんに以下を添えて連絡：
1. 何をしようとして
2. どこで詰まったか（画面のスクショ）
3. エラーメッセージ（あれば）
4. 使っているブラウザとOS
