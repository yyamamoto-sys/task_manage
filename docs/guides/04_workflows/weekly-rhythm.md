# 週次運用リズム

plan-app の OKR 機能を **チームで運用するときの1週間の流れ**です。
役割ごとに「いつ・何を・どこで」やるかを揃えるためのガイドで、メンバー研修資料としても使えます。

## 全体図

```
週初               週中              週末          翌週へ
 ─────────────── ───────────── ─────────── ──────►
 ①会議ノート      ②セッション    ③レポート     翌週①の
 （KR代表）        （合同）        （KR代表）    引き継ぎメモへ
 ＋自分のタスク     ＋宣言抽出      ＋確定        自動反映
   状態更新         （AI）
```

## 役割別の週次タスク（標準フォーマット）

### KR代表

| 曜日目安 | 何を | 場所 | 所要 |
|---|---|---|---|
| 週初 | **① 会議ノート**を記入（KR会議で／前に） | OKRモード → ①会議ノート | 15〜30分 |
| 週中 | ②セッションに**KRを代表して**参加・補強 | OKRモード → ②セッション記録&分析 | 1時間 |
| 週末 | **③ レポート**を AI 初稿→手修正→確定 | OKRモード → ③レポート作成 | 10〜20分 |
| 随時 | 自分の TF の TODO/タスク状態を更新 | 計画モード | 都度 |

### ファシリテーター

| 曜日目安 | 何を | 場所 | 所要 |
|---|---|---|---|
| 週初 | **チェックイン**の進行 | 会議体（Teams等） | 30〜60分 |
| 週初・終了直後 | 議事メモを ②セッション に投入 → AI分析 → 保存 | OKRモード → ② | 5〜10分 |
| 週末 | **ウィンセッション**の進行 | 会議体（Teams等） | 30〜60分 |
| 週末・終了直後 | 議事メモを ②セッション に投入 → AI分析 → 保存 | OKRモード → ② | 5〜10分 |

### 管理者

| 曜日目安 | 何を |
|---|---|
| 月初 / 期初 | AI使用量モニタを確認・メンバー追加削除 |
| クォーター末 | 次QのTF割り当てを確定（[四半期切替](./quarter-rollover.md)） |
| 随時 | 新規メンバー登録・PJ追加・トラブル対応 |

### TFメンバー（個別貢献者）

| 曜日目安 | 何を |
|---|---|
| 毎日 | 自分のタスクを「進行中／完了」に変える |
| 週初 | ダッシュボードで「今週の自分のタスク」を確認 |
| 週末（任意） | 完了タスクの contribution_memo を残す（後で振り返りや評価に使う） |

## 推奨タイミング例（参考）

実際の運用は各チームのリズムに合わせてください。以下は**典型例**です：

| 曜日 | 時間帯 | 誰 | 何を |
|---|---|---|---|
| 週初業務開始日 | 業務開始直後 | KR代表 | ① 会議ノート記入完了（KR会議実施 or 前日） |
| 週初業務開始日 | 業務開始2〜3時間後 | 全員 | チェックイン会議（合同） |
| 週初業務開始日 | チェックイン終了直後 | ファシリ | ②セッション保存 |
| 月〜金 | 随時 | 全員 | タスク状態更新 |
| 週末業務最終日 | 業務終了2〜3時間前 | 全員 | ウィンセッション（合同） |
| 週末業務最終日 | ウィン終了直後 | ファシリ | ②セッション保存 |
| 週末業務最終日 | ウィン後〜業務終了まで | KR代表 | ③レポート確定 |

## サイクルを切らさないための3つのルール

1. **③ レポートは "draft" のまま放置しない** — 確定（finalized）しないと翌週①の引き継ぎメモに反映されない
2. **② セッション保存は当日中に** — 議事メモは時間が経つと文脈が薄れる
3. **① 会議ノートは "事実" 欄に解釈を入れない** — AI 分析の精度が落ちる

## サイクル進捗バーの活用

OKRモードの上部にある **サイクル進捗バー**で、その KR が今週どこまで進んでいるかを一目で確認できます。

```
[KRタイトル] ｜5/12週   ● ①会議ノート（記入済）  ›  ○ ②セッション（未）  ›  ○ ③レポート（未）
```

進捗バーの各ステップをクリックすればそのタブに飛びます。**週末になってもバーが ①や② のままなら、サイクルが止まっているサイン**です。
