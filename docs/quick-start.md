# グループ計画管理アプリ クイックスタートガイド

> グループメンバー全員向けの入門ガイドです。  
> 最初の1週間でやることを中心に説明しています。

---

## アプリの基本的な考え方

このアプリは **OKR（目標管理）** の進捗をチームで共有・管理するためのツールです。

```
Objective（年間目標）
  └── KR（測定可能な成果指標）
        └── TF・Task Force（施策チーム）
              └── ToDo（やること）
                    └── Task（具体的なタスク）
```

全員が同じ画面でリアルタイムに進捗を見られます。

---

## 最初にやること（初回ログイン）

### 1. アプリを開く

グループ共有のURLにアクセスします。

### 2. 自分の名前を選択してログイン

名前のリストが表示されます。自分の名前をクリックしてください。

> パスワードは不要です。名前を選ぶだけでログインできます。

---

## 週次の使い方（定着するとここだけでOK）

### 月曜日：チェックイン（約15〜30分）

チェックイン会議をアプリと一緒に進めます。

**手順**

1. 左メニュー **「ラボ」→「KRセッション記録」** を開く
2. 対象KRを選択し、「チェックイン」モードを選ぶ
3. 会議の文字起こし・議事メモをテキストボックスに貼り付ける
4. 「AIで内容を抽出する」をクリック
5. 抽出された宣言・シグナルを確認・修正して保存

会議後、担当者はタスクボードで今週のタスクを更新します。

---

### 週中：タスク管理（毎日・数分）

**ダッシュボード** を開くと、自分の担当タスクと期限アラートが確認できます。

タスクの状態を更新するには：

1. **カンバン** または **リスト** 画面を開く
2. 担当タスクを見つけてステータスを変更（未着手 → 進行中 → 完了）
3. 期限が近いタスクはダッシュボードの「期限アラート」に表示されます

---

### 金曜日：ウィンセッション（約30分）

週の振り返り会議をアプリと一緒に進めます。

**手順**

1. 左メニュー **「ラボ」→「KRセッション記録」** を開く
2. 対象KRを選択し、「ウィンセッション」モードを選ぶ
3. 会議の文字起こし・議事メモを貼り付ける
4. 「AIで内容を抽出する」をクリック
5. 宣言達成状況（達成・部分達成・未達成）を確認・修正して保存

---

### 月1回（または必要に応じて）：KRレポート生成

**「ラボ」→「KRレポート生成」** から、会議内容をもとにHTMLレポートを自動生成できます。

1. 対象KRと会議種別（チェックイン／ウィンセッション）を選ぶ
2. 今週のセッションが記録済みの場合は「議事録を読み込む」ボタンが表示されます
3. 「AIでレポートを生成する」をクリック
4. 生成されたレポートをHTMLコピーして共有

## 画面一覧

| 画面 | 用途 |
|---|---|
| ダッシュボード | KR進捗サマリー・期限アラート・今週のタスク一覧 |
| カンバン | タスクをドラッグ＆ドロップで状態管理 |
| リスト | タスクを一覧・絞り込みで確認 |
| ガント | タスクのスケジュールをバーで可視化 |
| AI相談 | タスク整理・優先順位付けをAIと対話で実施 |
| 関係グラフ | OKR〜タスクのつながりを図で確認 |
| ラボ | KRセッション記録・KRレポート生成（AI機能） |
| 管理 | メンバー・OKR・TF・タスクの登録・編集 |

---

## タスクを追加するには

どの画面からでも、右下の **「＋」ボタン** からクイック追加できます。

| 項目 | 説明 |
|---|---|
| タスク名 | 具体的な行動レベルで書く（例：〇〇に電話して日程調整） |
| 担当者 | 自分または担当メンバーを選択 |
| 期日 | 完了予定日を設定 |
| ToDo紐づけ | どのKR施策に関連するかを設定（任意） |

---

## AI相談機能の使い方

左メニュー **「AI相談」** から、タスク整理や優先順位付けをAIと対話できます。

**よく使う質問例**

- 「今週優先すべきタスクを教えて」
- 「期限が近いタスクをまとめて確認したい」
- 「○○プロジェクトの進捗を整理して」

> AI相談はタスク・プロジェクト情報のみを参照します。OKR・KRの詳細情報はAIには渡りません。

---

## こまったときは

**「自分のタスクが表示されない」**
→ ダッシュボード右上の「全員」ボタンをクリックして、フィルターを解除してください。

**「KRセッション記録が保存できない」**
→ 議事メモの入力後、「AIで内容を抽出する」→ 内容を確認 → 「保存する」の手順を踏んでください。

**「レポートが生成されない」**
→ 議事メモが入力されているか確認してください。入力済みの場合はシステム担当者に連絡してください。
