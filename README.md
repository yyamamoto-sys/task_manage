# グループ計画管理アプリ（ローカルプロトタイプ）

チーム向け計画管理ツールのプロトタイプです。
現在はローカルストレージで動作します（Supabase承認後にDB移行予定）。

## セットアップ手順

```bash
# 1. 依存パッケージをインストール
npm install

# 2. 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:5173 を開くと起動します。

## 初回起動

- セットアップウィザードでメンバーを登録します
- 登録後、メンバー選択画面が表示されます
- 名前をクリックするとログインできます（次回以降は自動ログイン）

## 実装済みの機能

- ✅ セットアップウィザード（初回起動時のメンバー登録）
- ✅ メンバー選択（暫定ログイン）
- ✅ ダッシュボードビュー（KR進捗・今週タスク・アラート）
- ✅ カンバンビュー（ドラッグ&ドロップ、タスク追加）
- ✅ ガントビュー（タスクバー・今日線、タスク編集）
- ✅ リストビュー（フィルター・ソート・CSVエクスポート）
- ✅ タスク詳細・編集モーダル（論理削除）
- ✅ 管理画面（OKR / Task Force / プロジェクト / メンバー）
- ✅ ローカルストレージへのデータ永続化

## 実装予定の機能

- 🔲 AIに変更を相談パネル（Supabase Edge Function経由）
- 🔲 Supabase移行（IT部門承認後）
- 🔲 Microsoft Teams埋め込み対応

## メンバーの変更

管理画面（サイドバー「管理」）からメンバーの追加・編集ができます。
初期データは `src/lib/localData/seed.ts` で管理しています（`.gitignore` 対象）。

## 環境変数

`.env.example` をコピーして `.env.local` を作成してください。
Supabase承認前は設定不要です。

```bash
cp .env.example .env.local
```
